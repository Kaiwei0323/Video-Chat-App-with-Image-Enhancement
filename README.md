# Video-Chat-App
Implement super-resolution algorithm to achieve real-time streaming data resolutioin recovery

## Prerequisites
```
  Python 3.7
  Tensorflow 2.10.0
  PyTorch
  OpenCV 4.7.0.72
  Flask
  
```

## Model
```
  ESCPN
  ESRGAN
  PyTorch-Rcan
  PyTorch-SRResNet
  SRRENET
```
